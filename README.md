### Turbo.az data analysis & Power BI Dashboard

Bu layihədə Turbo.az saytından toplanmış avtomobil elanlarının datası Python vasitəsi ilə təmizlənib və analiz edilib. Daha sonra nəticələr Power BI istifadə edilərək interaktiv dashboard şəklində vizuallaşdırılıb. 

### Layihənin məqsədi

- Azərbaycan avtomobil bazarının ümumi vəziyyətini analiz etmək
- Qiymət paylanmasını araşdırmaq
- Marka və modellərə görə statistik göstəriciləri çıxarmaq
- İl və mühərrik həcminə görədəyişiklikləri təhlil etmək

### Analizlər

- Yanacaq növünə görə paylanma
- Buraxılış ili üzrə qiymət dəyişimi
- Ban növü üzrə qymət dəyişimi
- Ötrücü növü üzrə qiymət dəyişimi
- Şəhərlərə görə elan sayı
- Rəng üzrə elan sayı



### Power BI Dashboard

Dashboard aşağıdakı bölmələrdən ibarətdir:

- Ümumi statistika (KPI-lar)
- Model üzrə filter
- Qiymət üzrə analiz
- Şəhər, rəng və yanacaq növü üzrə analiz
